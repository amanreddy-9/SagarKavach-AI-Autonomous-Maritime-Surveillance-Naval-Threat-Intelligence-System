## 🌊 Project Name Meaning — SagarKavach

The name **SagarKavach** is derived from two Sanskrit words:

- **Sagar (सागर)** — Ocean / Sea  
- **Kavach (कवच)** — Shield / Protection / Defense Armor  

Together, **SagarKavach** translates to **"Ocean Shield"** or **"Maritime Defense Shield"**.

The name reflects the core objective of the project — building an **AI-powered intelligent defense layer for maritime environments**. Just as a shield protects against threats, **SagarKavach-AI acts as a digital surveillance shield over oceans**, continuously monitoring naval activity, detecting potential threats, and providing real-time situational awareness.

SagarKavach-AI functions as:

# 📌 Project Overview

SagarKavach-AI is an **AI-powered maritime surveillance and naval threat intelligence system** designed to automatically detect, classify, and analyze naval warships using deep learning and real-time computer vision. The system leverages the **YOLOv8-OBB (Oriented Bounding Box)** architecture to accurately detect naval vessels in both images and live video streams, even under challenging maritime conditions such as varying lighting, ocean clutter, scale variations, and rotational orientations.

Unlike traditional monitoring approaches that rely on manual observation or rule-based detection, SagarKavach-AI adopts a **fully data-driven deep learning framework** capable of identifying multiple classes of naval vessels, including:

- Aircraft Carriers  
- Destroyers  
- Corvettes  
- Submarines  

The use of **Oriented Bounding Boxes (OBB)** significantly improves localization accuracy, particularly for elongated vessels captured at arbitrary angles in aerial and satellite imagery. This makes the system highly effective for **real-world maritime surveillance scenarios**, where ships rarely appear in fixed orientations.
